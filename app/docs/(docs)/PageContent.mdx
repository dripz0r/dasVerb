## What is DasVerb?

DasVerb is a next-generation language learning platform that makes learning German fun, engaging, and effective. Built with modern technologies like Next.js, TypeScript, and AI, it offers:

- Interactive lessons with real-time feedback
- AI-powered conversation practice
- Smart flashcards with spaced repetition
- Progress tracking and gamification
- Modern, responsive design

## Tech Stack

- **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS
- **UI Components:** ShadCN UI
- **AI Integration:** Google Gemini
- **Backend:** Genkit (Firebase)
- **Styling:** Tailwind CSS
- **Documentation:** Nextra
