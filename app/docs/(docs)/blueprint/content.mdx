# **App Name**: DeutschTok

## Core Features:

- Landing Page: Landing page with a Gen Z-focused tagline, feature highlights, and clear CTAs.
- User Authentication: User authentication using Firebase Auth.
- Dashboard: Personalized dashboard displaying progress, streaks, and recommended lessons.
- Interactive Lessons: Dynamic lesson pages with grammar explanations, quizzes, and fill-in-the-blanks.
- Vocab Flashcards: Themed vocabulary decks with SRS flashcards, audio, and GIFs.
- Grammar Fixer: AI-powered grammar correction tool with explanations.
- AI Story Generator: Story Generator:  Uses learned vocab into various story modes (fairy tale, TikTok drama, etc.); output of this tool might occasionally contain ironic GenZ references to provide extra flair for students.

## Style Guidelines:

- Primary color: Vibrant purple (#A020F0) to capture the energy and creativity of Gen Z.
- Background color: Light lavender (#E6E6FA) to provide a soft and modern feel.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights.
- Body and headline font: 'Space Grotesk' sans-serif, for a techy and modern feel.
- Use bold, modern icons, complemented with occasional reaction GIFs and emojis.
- Mobile-first design with a clear, structured layout inspired by Notion, and bold progress bars
- Fun, quick animations on interactions, gamified learning, and lesson completions.
